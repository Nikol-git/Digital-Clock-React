## Digital Clock (React)
A simple digital clock built with React, updating every second using useState and useEffect.
This component displays the current time in HH:MM:SS format.

## Features
- Real-time updating clock (updates every second)
- Built using React functional components
- Uses hooks (useState, useEffect)
- Automatically shows current system time
- Easy to style and integrate into any React project

## How It Works
1. The component stores the current time in state.
2. A setInterval updates the state every second.
3. React re-renders the component, showing the updated time.
4. Cleanup prevents memory leaks.
